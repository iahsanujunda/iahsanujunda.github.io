The first analysis that could be done on analytics project, is exploratory analysis. The main purpose of exploratory data analysis is to see if our data is appropriate for our question. <!-- more --> When engaging in exploratory data analysis, we need to constantly ask two questions, do we have the right data, and given what we see from the data, do we have the right question. Exploratory analysis is iterative process in itself, it is not a one way journey. So when we start by looking at the question, we might find that our data can not help us answer this question, then we might need to get another data. Or if the data pass this qualification, but we find that the question is lacking in some way, then we might need to reformulate the question.
