Movie Review Sentiment Analysis

PyTorch | NLP | React | AWS
LSTM recurrent neural network for sentiment analysis that is trained on IMDB dataset and developed using pytorch. The model is deployed on AWS sagemaker and can be invoked from AWS lambda. Public access to the model is established from React web app that will send http request to invoke the lambda function.

Machine Learning on Sagemaker

A series of case studies on how to do machine learning engineering in AWS, covering unsupervised learning, supervised learning, and time-series analysis. These case studies are capped with a project on plagiarism detection that combine several techniques from earlier parts.
